# Unity 太空射擊遊戲專案

---

## 一、專案簡介

本專案為一個以 Unity 開發的三維太空射擊遊戲，屬於學生專題作品。玩家需操作太空飛船，自由的進行射擊與移動，達到生存與戰鬥的遊戲體驗。

本專案著重於遊戲核心機制的實作，包含玩家控制 射擊系統及音效整合，並設計完整的遊戲循環，使其具備可遊玩的基本原型。

此外，專案中亦包含 Android 使用者介面設計，作為未來跨平台整合的延伸方向。

---

## 二、系統架構說明

### （一）遊戲核心系統

本專案以 Unity 為主要開發平台，透過場景與遊戲物件管理實現以下功能：

* 玩家飛船控制與移動
* 射擊系統與子彈生成
* 音效與背景音樂播放

此部分構成完整的遊戲主體。

---

### （二）音效與視覺效果

專案中整合多種音效與特效資源：

* 背景音樂用於提升沉浸感
* 射擊音效增加操作回饋

使遊戲不僅具備功能性，也具備基礎的視覺與聽覺體驗。

---

### （三）Android 延伸設計

專案中包含 Android 使用者介面程式碼，例如：

* 登入畫面
* 難度選擇畫面
* 遊戲結束畫面

此部分作為應用程式整合的延伸設計，目前尚未與 Unity 遊戲流程完成整合，但顯示專案具備跨平台應用開發的方向。

---

## 三、遊戲流程說明

玩家進入遊戲後將直接進入主場景，進行以下流程：

1. 透過 ( w a s d ) 控制飛船移動
2. 使用滑鼠左鍵發射子彈攻擊
3. 持續生存並應對挑戰

形成基本的即時操作型射擊遊戲體驗。

## 九、結論

本專案成功實作一個具備完整遊戲循環的太空射擊遊戲，並透過 Unity 建立核心玩法與互動機制。
