專案名稱
Unity 太空射擊遊戲

專案簡介
本專案為一個以 Unity 開發的三維太空射擊遊戲，屬於學生專題作品。玩家在遊戲中操作太空飛船，透過移動與射擊來破壞敵人與隕石，並在不斷增加的威脅中生存。專案重點放在遊戲核心玩法的實作，包含移動控制 射擊系統 碰撞判定與音效整合，呈現一個完整可遊玩的遊戲原型，同時具備擴充為完整應用程式的設計基礎。

專案亮點
具備完整可遊玩的射擊遊戲核心循環
包含玩家控制 射擊判定 敵人生成與碰撞回饋
整合爆炸特效 雷射效果與音效 提升臨場感
專案結構清楚 區分資源 腳本與場景
包含 Android 使用者介面設計作為延伸開發方向

遊戲內容說明
玩家透過控制飛船在畫面中移動，並發射武器攻擊敵人
場景中會持續生成敵方飛船與隕石，增加遊戲難度與壓力
當物件發生碰撞時會觸發爆炸效果與音效，提供即時回饋
遊戲過程中搭配背景音樂，提升整體沉浸感與節奏感

系統設計說明
本專案採用 Unity 作為主要遊戲引擎，透過場景管理與物件控制實現遊戲邏輯
玩家物件負責接收輸入並進行移動與射擊
敵人物件具備生成與移動邏輯，持續對玩家造成威脅
碰撞系統負責判定子彈與敵人或玩家之間的互動，並觸發對應效果
音效系統負責播放射擊與爆炸聲音，以及背景音樂

專案結構說明
Assets 資料夾包含所有遊戲資源
模型資源包含玩家飛船 敵方飛船與隕石物件
音效資源包含射擊聲 爆炸聲與背景音樂
材質與特效資源用於呈現雷射與爆炸效果

Scripts 資料夾包含 C Sharp 腳本
負責玩家控制 射擊邏輯 碰撞處理與遊戲流程控制

Scenes 資料夾包含遊戲場景
主要場景為 SampleScene 執行後會直接進入遊戲

Android 延伸功能說明
專案中包含 Android 使用者介面相關程式碼，例如登入畫面 難度選擇與遊戲結束畫面
此部分作為遊戲系統的延伸功能進行設計，目前尚未完成與 Unity 遊戲流程整合
顯示專案具備跨平台應用整合的設計方向

使用技術
Unity 遊戲引擎
C Sharp 腳本開發
三維模型與材質應用
音效與背景音樂整合
Android Java 使用者介面設計

操作方式
開啟 Unity 專案並載入 SampleScene
執行後即可開始遊戲
使用鍵盤控制飛船移動
進行射擊並閃避敵人與隕石
