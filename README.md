專案名稱
Unity 太空射擊遊戲

專案簡介
本專案為一個以學生專題形式完成的三維太空射擊遊戲，使用 Unity 開發。玩家在遊戲中操作飛船進行射擊並躲避敵人與隕石，透過即時操作與反應提升遊戲體驗。本專案目前以遊戲核心玩法為主，著重於遊戲機制與互動設計。

開發動機
本專案旨在學習 Unity 遊戲開發流程，包含物件控制 碰撞判定 與基礎遊戲設計，並透過實作提升對遊戲系統架構的理解。

功能介紹
玩家可控制飛船進行移動
具備射擊功能與攻擊效果
敵人與隕石會出現在場景中增加挑戰
包含爆炸效果與音效提升臨場感
提供背景音樂增加遊戲沉浸感

系統說明
本專案主要由 Unity 建構遊戲場景與邏輯
遊戲啟動後會直接進入主場景進行遊戲
專案中曾規劃加入 Android 使用者介面例如登入與難度選擇，但目前尚未完成整合

使用技術
Unity 遊戲引擎
C Sharp 腳本開發遊戲邏輯
三維模型與音效素材應用

操作方式
開啟 Unity 專案並執行場景後即可開始遊戲
使用鍵盤控制飛船移動
進行射擊並躲避敵人與隕石

學習成果
熟悉 Unity 基本操作與遊戲開發流程
理解遊戲物件控制與碰撞處理
學習音效與視覺效果整合
提升問題解決與專案實作能力

未來改進方向
完善使用者介面系統
加入登入與難度選擇功能並完成整合
增加分數系統與遊戲目標
設計更多關卡與敵人類型
優化操作手感與畫面效果

作者
